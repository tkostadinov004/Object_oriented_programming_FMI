## Задачи

**1.** В езици като Java низовете се съхраняват в така наречения *string pool*. Най-просто казано, това представлява колекция от низове. При създаване на низ,
се проверява дали такъв низ вече се съдържа в string pool-a и ако такъв има, се връща указател към него, вместо да се заделя допълнителна памет. Ако такъв няма, се добавя в колекцията.
Например,
```c++
ImmutableString a = "test";
ImmutableString b = "test";

// тук низовете в a и b сочат към една и съща памет 
```

Реализирайте клас *ImmutableString*, който представлява низ в string pool. 
Реализирайте клас *StringPool*, който е singleton и реализира следните функционалности:
- добавяне на срещане на низ в колекцията
- проверка дали низ се съдържа в колекцията
- премахване на низ от колекцията

**2.** Реализирайте прототип на файлова система и интерфейс за работа с нея. 
Всеки файл има име и атрибути (дали е скрит, дали е архивируем, дали е read-only, дали е системен)

Има два вида файлове - *обикновени файлове* и *директории* (която може да съдържа други директории или обикновени файлове). Файловата система е представена
като дърво, в което вътрешни върхове могат да бъдат само директории.

Реализирайте следните команди за работа с файловата система:
- добавяне на файл в текущата директория
- премахване на файл от текущата директория (ако файлът е директория, изтрива рекурсивно)
- смяна на текущата директория (аналог на *cd* в bash и powershell)
- записване на всички архивируеми файлове в текстов файл
