**13.** Нека имаме следната структура:
```c++
template<typename T>
struct DynamicArray {
    T* data;
    size_t size;
};
```

Структурата представлява масив с точно size на брой елементи, съхранявани в динамично заделена памет.

Напишете шаблони на функции:
```c++
template<typename T, typename U>
DynamicArray<U> map(U(*f)(const T&), const DynamicArray<T>& arr);
```
която приема едноместна функция f и динамичен масив arr, връща нов динамичен масив, получен след поелементното прилагане на функцията f върху елементите на arr.

```c++
template<typename T>
DynamicArray<T> filter(bool(*p)(const T&), const DynamicArray<T>& arr);
```
която приема едноместен предикат p и динамичен масив arr, връща нов динамичен масив, съдържащ само онези елементи на arr, за които p(element) е истина.
